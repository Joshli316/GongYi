## Key Findings for GongYi Build

1. **No direct competitor** — No free, bilingual, privacy-first OPT compliance tracker exists. Chinese-language support is virtually absent despite Chinese nationals being the largest F-1 population.
2. **Unemployment counting is the #1 pain point** — Students have no easy way to calculate days; SEVIS counter is DSO-only.
3. **Data sources are available but fragmented** — CIP codes from NCES (pipe-delimited text), STEM designation from DHS (PDF-only, must parse), E-Verify has no public API (must use search tool or DOL LCA proxy).
4. **10-day reporting deadlines** are frequently missed — a reminder/calendar system addresses a real gap.
5. **I-983 tracking** is unserved — no existing tool tracks evaluation deadlines or material change triggers.
6. **Cap-gap rules** changed January 2025 (extended to April 1) — most guides are outdated.
